This directory contains an exported static html copy of my wordpress blog

It is configured within a git repository with alocal repo in this directory, with the remote at: https://github.com/v3rt3x-4rr4y/sjd-blog.git

It is also being served as a static web site on GitHub pages at https://v3rt3x-4rr4y.github.io/sjd-blog/

To push to any changes made locally to the corresponding remote git repo, use these git bash commands:

git commit -m "<commit note>"
git push origin
